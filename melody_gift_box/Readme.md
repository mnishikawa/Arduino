■　作り方等の情報

秋月に電子オルゴール用のメロディICってのが売っていて、電池とスピーカーをつなげるとかんたんにメロディを流すことができます。しかも5個で150円とかかなりお買い得です。メッセージカードで音楽が流れるものがありますが、これをお手軽にいろんなシーンで活用することができます。プレゼントボックスを開けた瞬間にバースデーソングが流れる箱を作ってみました。
